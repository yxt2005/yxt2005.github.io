搭建个人博客

2025-08-07 学习笔记 Hexo, 博客搭建 Comments

我们在学习过程中，尤其是碎片化的学习中，经常需要通过写一些博客记录学习过程。因此，搭建一个简洁的个人博客网站可以让我们的学习更加体系。GitHub 作为全球知名的开源代码托管平台，为我们提供了稳定且免费的存储服务，而 Hexo 作为一款基于 Node.js 的快速、简洁、高效的静态博客框架，两者结合，为我们打造个人博客带来了极大的便利，能让我们轻松搭建出既美观又功能丰富的博客。

(1) 准备工作

  1. 安装Node.js

    • 官网下载：https://nodejs.org/

    • 验证安装：

  2. 安装Git

    • 官网下载：https://git-scm.com/
    • 验证安装：git --version

  3. 注册github账号

（2）安装hexo

  1. 本地创建博客目录：myblog

  2. 安装Hexo:npm install -g hexo-cli

  3. 初始化博客：

    • 博客目录下运行：hexo init
    • 相关文件：
      • _config.yml：站点配置文件：用于修改主题和部署配置
      • source：博客文章和页面源文件
      • themes: 主题
      • **public:**静态文件，部署到github上，通过浏览器访问

  4. 选择主题，修改配置文件

    • 主题选择：官网https://hexo.io/themes/，我选择的是pure
    • 在博客目录下的命令行中运行git clone https://github.com/cofess/hexo-theme-pure themes/pure
    • 在站点配置文件中，在_config.yml文件中修改theme的值为对应值pure
    • 在主题配置文件中：修改站点名、作者名称、头像、自我介绍等

（3）创建博客

  1. 博客目录下运行：hexo new "文章标题"
  2. 使用文本编辑器或Typora编写
  3. 本地预览：hexo s

(4) 将博客部署到github

  1. 创建github仓库：

    • 仓库名：用户名.github.io(必须)
    • 仓库类型：public

  2. 安装hexo-deployer-git

    npm install hexo-deployer-git --save

  3. 配置SSH秘钥（如果后面要使用SSH）

    • 配置SSH key:

    • 查看秘钥：cat ~/.ssh/id_rsa.pub

    • 添加主机到SSH可信列表：ssh -T git@github.com,出现提示即为连接成功Hi yxt2005! You've successfully authenticated...

    • 登录github添加SSH key:主页头像->settings->SSH and GPG keys,添加刚刚生成的的秘钥
